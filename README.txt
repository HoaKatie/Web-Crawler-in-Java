WEB CRAWLER AND SEARCH ENGINE IN JAVA AND JAVAFX

Instructions to run:
Run SearchResultApp.java

Using JavaFx to simulate the GUI of the search engine using Model View Controller (MVC) to return the most relevant results to least in descending order. 
The calculations of returning relevant results after crawling is through page rank algorithm using vector space model and cosine similarity.

The crawler and search also handle exceptions while handling buffer for file in/file out.
